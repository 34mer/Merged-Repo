# Invariant

## Terminal object
Continuity-preserving substrate migration of a live self-maintaining center.

## Non-substitutable rule
Do not replace the object with:
- human enhancement
- medical restoration as terminal aim
- copied memory
- copied behavior
- functional equivalence alone
- generic whole-brain emulation treated as automatically sufficient
- generic AGI progress
- institutional survival
- prestige accumulation

## Core distinction
Substrate viability opens the project.
Continuity-valid migration defines the project.

## Current doctrine
Continuity tracks one live, causally continuous, self-maintaining regulatory process remaining in operation while burden is progressively displaced across substrate.
