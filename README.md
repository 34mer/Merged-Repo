# ProjectX Merged Repo

This repo binds three layers:

- `projectx/`: outer kernel, runtime state, control, eval, and continuity surfaces.
- `pg_pipeline/`: typed positive-geometry research lifecycle.
- `exact_work/`: imported mathematical authority, handoff notes, Lean scaffold, and exact artifacts.

The first runnable target is intentionally narrow: generate one small family, normalize records, propose a recurrence, check it, stress it, compile a substrate draft, and emit a weekly-style summary.

The second runnable target carries the current thread's realization-class work into the typed lifecycle: generate the R2↔R3 comparison record, check its structural shape, store authority-linked records, and emit a comparison summary.

The supervised ProjectX run also imports source-backed authority records from the research-ledger Lean/handoff files.
