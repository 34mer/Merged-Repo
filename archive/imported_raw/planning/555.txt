Exact “done” criteria for each file cluster Schemas are done when: • every record validates, • extra fields are rejected, • JSON roundtrip works. Storage is done when: • init_db.py creates tables, • records insert/fetch cleanly, • duplicate IDs are blocked. Generator is done when: • it emits at least 5 exact records, • held-out cases can be generated later. Proposer path is done when: • prompt builds, • model or mock emits valid JSON, • invalid output is rejected/retried. Checker path is done when: • one valid proposal passes, • one invalid proposal fails, • failure record is stored. Stress layer is done when: • one proposal passes held-out, • one broken mutation causes fail. Substrate compiler is done when: • a valid schema record is emitted from a strong survivor.   ⸻   21.
