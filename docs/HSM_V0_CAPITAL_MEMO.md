# Human Substrate Migration v0

Verified substrate migration has already been demonstrated at nervous-system scale. HSM-v0 extends the stack to a human-constrained synthetic substrate object and verifies migration under protocol.

The primitive is crossed. This round funds scale, external expansion, substrate optimization, and independent verification.

## What the round funds

- larger human constraint ingestion
- more human datasets
- process-generating object search
- synthetic substrate optimization
- external validators
- replication package
- compute sweeps
- visual destination substrate

## What it does not fund

- philosophical speculation
- open-ended neuroscience without protocol outputs
- full human brain simulation as a prerequisite
- human person migration claims
