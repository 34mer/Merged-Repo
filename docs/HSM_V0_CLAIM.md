# HSM-v0 Claim

This project demonstrates that human substrate migration is empirically possible under the HSM-v0 protocol. A human-constrained substrate object is constructed from verified primitive chains and external human neural constraints, translated into a synthetic substrate, migrated, re-instantiated, and verified.

This is not a claim that a human person has been migrated. It is a constructive empirical demonstration that a human-constrained substrate object can undergo the migration operation.
