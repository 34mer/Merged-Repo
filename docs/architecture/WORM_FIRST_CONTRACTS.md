# First Contracts

As money comes in through an approved legal structure, convert attention into contracted work.

## Contract A: biological protocol design with a worm lab

Define experimental animals, recording conditions, preservation/readout candidates, re-embodiment target, success criteria, failure criteria, and data release rules.

## Contract B: behavior fingerprinting system

Build repeatable individual behavior recording and analysis for baseline, perturbation, and post-transfer comparison.

## Contract C: preservation/readout feasibility

Assess readout methods that could preserve individual-specific structure relevant to behavior modeling.

## Contract D: individual-specific modeling pipeline

Build the computational pipeline that maps individual readout and behavior data into a runnable digital or synthetic-body controller candidate.

## Contract E: validation standard

Define blind tests, negative controls, statistical thresholds, audit logs, independent verification, and public reporting rules.
