# HSM-v0 Scope

HSM-v0 proves one narrow operation: a human-constrained synthetic substrate object can be constructed, migrated, resumed, and verified under protocol.

It does not claim:

- human person migration
- consciousness transfer
- full brain simulation
- full connectome emulation
- exact voltage-trace copying
- biological upload
- scanner sufficiency

The target object is a deterministic, checkpointable, hashable, human-constrained substrate object.
